About

Hello there, I'm Tal Korman, Student at HackerU college, studying Mobile apps development for Android and IOS. My coding jurney began at the 80s, while practicing coding games in BASIC language on Dragon 64 computer and Apple 2E, that was a great time for developing Karate, Space and Quest games from scratch. At high school I learned Pascal software and continued studying Electronics Practical Engineering, graduated with a final project of programed automotive robot for delivering objects from one place to another, coded with Assembly language on 8051 CPU. Many years passed and on January 2019 I took some online courses on Javascript language and developed a complicated app for Loading containers (see Loading Optimizator at gallery). Recently I've completed another Javascript course at HackerU college which helped me to improve and increase my web coding and design skills. Currently I study Java at HackerU and looking forward to develop new ideas of apps on mobile devices.

Loading Optimizator

I'm excited to introduce to you my first Javascript app, ...following an old idea that came up while calculating and loading containers in China, struggeling on how to insert the total quantity of the goods to a container with minimal wasted spaces. The app will do for you the frustrating job and provide you the best configuration of how to load your consignment. The app's AI logic which was my top developing challange so far is doing an imitation of the human logic for achieving the same task. On the current version for a security reason I moved the AI module from the browser side to a server with Node js and Express js. I used Threejs library for rendering the 3D scene and developed a mouse and touch mechanism control for orientation in 3 dimension enviroment which was a bit tough geometrical challange to achieve a high level of smoothness orientation. The styling was modified to fit variety of screen sizes as well as mobile devices. The software is still been developed and more features are planned to be added...
